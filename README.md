# Startup_landing_webpage
🚀 NovaAI Landing Page

A modern and responsive startup landing page built using HTML and CSS.
This project showcases a clean UI design for an AI-based startup website with sections like features, testimonials, call-to-action, and contact form.

🌟 Features
Responsive Navigation Bar

Hero Section with CTA Button

Features Section

Testimonials Section

Contact Form UI

Modern Responsive Design

Smooth Hover Effects

Social Media Footer Icons


🛠️ Technologies Used
HTML5

CSS3

Bootstrap Icons

Google Fonts
